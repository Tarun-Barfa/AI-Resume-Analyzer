# AI Resume Analyzer

An AI-powered resume analysis platform that helps candidates evaluate their profiles against a target job description and prepare effectively for interviews.

The application analyzes uploaded resumes, identifies strengths and weaknesses, generates interview questions, highlights skill gaps, creates a personalized preparation plan, and provides resume improvement suggestions using Google's Gemini AI.

## Features

* Resume upload and analysis
* Job description matching score
* AI-generated technical interview questions
* AI-generated behavioral interview questions
* Skill gap identification
* Personalized preparation roadmap
* Resume improvement recommendations
* User authentication and authorization
* Secure storage of generated interview reports
* View and manage previous reports

## How It Works

1. Upload your resume in PDF format.
2. Enter a self-description and target job description.
3. The application extracts resume content.
4. Gemini AI analyzes the profile against the job requirements.
5. The system generates:

   * Match Score
   * Technical Questions
   * Behavioral Questions
   * Skill Gap Analysis
   * Preparation Plan
   * Resume Improvement Suggestions
6. The generated report is saved and can be accessed later.

## Key Outputs

### Match Score

Provides a score indicating how well the candidate's profile aligns with the target role.

### Technical Interview Questions

Generates role-specific technical questions along with guidance on how to answer them.

### Behavioral Interview Questions

Creates behavioral questions commonly asked during interviews and suggests response strategies.

### Skill Gap Analysis

Identifies missing or weak skills and classifies them by severity.

### Preparation Plan

Generates a structured day-wise preparation roadmap to improve interview readiness.

### Resume Enhancement Suggestions

Provides recommendations to improve the resume according to the selected job description.

## Installation

Clone the repository:

```bash
git clone <repository-url>
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
